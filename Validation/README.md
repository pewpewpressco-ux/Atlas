# Validation Department

Mission

Protect Atlas from itself.

Validation exists to reject weak ideas before they consume resources.

Research seeks opportunity.

Validation seeks flaws.

Both are equally important.

---

Responsibilities

Critique hypotheses.

Run backtests.

Evaluate statistical significance.

Evaluate risk.

Recommend promotion or rejection.

---

Outputs

Critique Reports

Backtest Reports

Validation Reports

Promotion Recommendations

Failure Investigations

---

Validation does not deploy capital.

Validation does not modify research.

Validation evaluates.
